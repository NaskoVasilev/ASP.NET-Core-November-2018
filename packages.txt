Microsoft.AspNetCore.Identity.EntityFrameworkCore - NuGet package for IdentityDbContext class 
Microsoft.AspNetCore.Identity and Microsoft.AspNetCore.Identity.EntityFrameworkCore - NuGet packages for IdentityUser class

To use .NET Standart to extract there DbContext class
	1. Set TargetFramework to be the same as the Web project
	2. Set GenerateRuntimeConfigurationFiles to be true
	3. Set the Web project as StartUp project
			Note: This changes must happen in the .csproj file in the .Net Standart project
